Gan Keyboard  
(c) freedom-blind  

## Description  
This is a Keyman keyboard for the Gan Chinese language, optimized for the Xianning (咸宁) regional variety of Gan.

It supports a six-tone system designed for phonetic clarity and practical typing use.

Tone input is mapped as follows:  
1st tone = j  
2nd tone = r  
3rd tone = d  
4th tone = f  
5th tone = k  
6th tone = l
